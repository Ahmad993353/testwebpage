# WebPage Admin

#### Prerequisites
  - Node.js
  - NPM
  - Git

#### Installing & Local Development
Start by typing the following commands in your terminal

```
> git clone https://github.com/Ahmad993353/testwebpage/tree/main/testwebpage webpage
> cd webpage
> npm init
> npm install express
> node server.js

```
